# Gmail × Amazon SES 送信設定

## 概要

Gmail から **Amazon SES 経由**でメールを送信するための設定手順です。

- **ユーザー名 / パスワード**
  - 他のメールアドレスで **Amazon SES による送信設定が済んでいる場合** → 同じ認証情報を使う
  - **まだ設定していない場合** → 管理者にアカウント発行を依頼する
- 利用者は **Gmail 側の設定のみ** 実施

---

## 設定手順

### 1. Gmail の設定画面を開く

右上の **歯車アイコン** → **「すべての設定を表示」** をクリック

### 2. 「アカウントとインポート」タブを開く

### 3. 「他のメール アドレスを追加」をクリック

### 4. 追加したいメールアドレスを入力

### 5. 「エイリアスとして扱います」にチェック

### 6. 「次のステップ」をクリック

### 7. SMTP 情報を入力

| 項目 | 設定値 |
|---|---|
| SMTP サーバー | `email-smtp.ap-northeast-1.amazonaws.com` |
| ポート | 587 |
| ユーザー名 | 既存の SES 設定と **同じ認証情報**（未設定の場合は管理者より発行） |
| パスワード | 既存の SES 設定と **同じ認証情報**（未設定の場合は管理者より発行） |
| セキュリティ | **TLS を使用したセキュリティで保護された接続** を選択 |

### 8. 「変更を保存」をクリック

### 9. 確認メールで認証

Gmail に確認メールが届くので、メール内の URL をクリックして認証を完了してください。

---

## 注意事項

- SMTP 認証情報は **個人専用（共有禁止）**
- テスト目的で **ランダムなメールアドレスへ送信しない**
- 不要になった場合は **管理者へ削除依頼**
